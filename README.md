## Настройка пакетного менеджера и окружения
1) Скачай uv через ```pip install uv``` или ```curl -LsSf https://astral.sh/uv/install.sh | sh```
2) Скопируй репозиторий к себе
3) Запусти команду ```uv sync``` - он скачает все зависимости, создаст и запустит venv
4) Запусти сервер ```uv run uvicorn main:app```
5) Сначала он будет загружать модель, немного надо подождать

## Endpoint'ы
1) health - проверяет статус, работает ли сервис
2) ask - задаешь вопрос, указывая 2 параметра book_name и query
3) book_name = ['manas', 'semetey', 'syngan_kylych', 'uzak-jol'] - выбираешь один из них и указываешь также
4) query = {задаешь вопрос}
